El concepto de “Web 2.0” comenzó con una sesión de lluvia de ideas realizada entre O'Reilly y MediaLive International, en donde se observó que lejos de colapsarse, la Web estaba tomando un rol más importante, con innovadoras nuevas aplicaciones y con sitios Web apareciendo con sorprendente regularidad.

La lluvia de ideas para definir el término partió de la comparación de una serie de servicios disponibles en ese momento que se ejemplifica con la siguiente tabla.

La Web 2.0 es también llamada Web Social por el enfoque colaborativo y de construcción social de esta herramienta

En las siguientes imágenes se ilustran las diferencias entre Web 1.0 y Web 2.0.

Haz clic en las ilustraciones para revisar su descripción.
Web 1.0

En la web 1.0 una persona (webmaster) colocaba la información en un sitio web y todos los usuarios la consultaban, siendo el webmaster el responsable de lo publicado.

Los usuarios solo sabian que se actualizó esa información cuando ingresaban y veían la diferencia o bien si se les informaba a través de un correo electrónico o cualquier medio digital.

Web 2.0

En la Web 2.0 no solo una persona coloca información, sino un grupo de usuarios puede publicar o construir la información que está en los sitios web a través de diferentes herramientas, de tal forma que si existe una actualización por cualquier usuario se le aviso a los demás de forma electrónica a través de un RSS de forma automática.
